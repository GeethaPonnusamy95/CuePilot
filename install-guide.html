Important Security Notice

CuePilot is currently an unsigned application. This means macOS will show security warnings when you try to install it. This is normal for indie apps that haven't gone through Apple's expensive code signing process.

Why unsigned? Code signing certificates cost $99/year and require Apple Developer membership. As a beta/free app, we're providing CuePilot unsigned to keep it accessible to everyone.
