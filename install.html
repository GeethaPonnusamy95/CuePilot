Installation Guide

macOS Installation

1 Download the App

Download the appropriate version for your Mac:

→ Apple Silicon (M1/M2): Download ARM64 version → Intel Processor: Download Intel version

2 Security Warning

"CuePilot" cannot be opened because the developer cannot be verified.

This is normal for open-source applications. Follow the next steps to proceed with installation.

3 Allow App Installation

  1. Open System Settings on your Mac
  2. Go to Privacy & Security
  3. Scroll down to Security
  4. Look for the message about CuePilot
  5. Click Open Anyway

4 Final Confirmation

Click Open in the final confirmation dialog to launch the app.

The app will now be saved as a trusted application on your system, and you won't see these warnings again.

System Requirements

  • macOS 11.0 or later
  • Compatible with both Intel and Apple Silicon Macs
  • Minimum 4GB RAM recommended
