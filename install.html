Installation Steps

1 First Launch

After downloading, double-click the CuePilot app. You might see this security message:

"CuePilot" cannot be opened because the developer cannot be verified.

This is normal for open-source applications and only appears on first launch.

2 Security Settings

  1. 1 Open System Settings on your Mac
  2. 2 Go to Privacy & Security
  3. 3 Scroll down to Security
  4. 4 Look for the message about CuePilot
  5. 5 Click Open Anyway

3 Final Step

Click Open in the final confirmation dialog to launch CuePilot.

🎉 That's it! CuePilot is now saved as a trusted application on your system. You won't see these security prompts again.
